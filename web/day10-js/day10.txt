2.函数
	1.什么是函数
		函数,即function,是一段预定义好,可以独立执行并包含多条执行语句的代码块
		优点:
			1.多条执行语句的封装
			2.实现代码的复用
	2.在JS中创建函数
		function 函数名(参数列表){
			函数体
			[返回值]
		}
	  参数列表:
	  	如果没有参数,可以为空
	  	如果有参数,就编写参数列表,如果参数是多个的话,各个参数之间使用","隔开
	  	有参数的函数,在调用时就要传参,如果未传参的话,参数的值就是undefined

	  ex1:
	  	function show(){

	  	}
	  ex2:
	  	function show(name,age){

	  	}
	  返回值:
	  	在函数体内,经过运算后,需要传递给函数调用者的一个值,就是返回值
	  	返回值是可选的,有返回值的话,需要使用return进行,
	  	返回,最多只有一个值

	4.变量的作用域
		1.什么是变量的作用域
			变量的作用域指的是变量的可访问范围
		2.作用域的分类
			1.局部变量
				使用var关键字声明在某个函数内的变量,都是局部变量

				局部变量只能在声明的某个函数内使用,出了函数则无法使用
			2.全局变量
				1.在function之外的声明变量都是全局变量
				2.声明变量不使用var关键字的时候,都是全局变量

				全局变量可以应用在各个函数以及各个位置
				注意:全局变量,推荐放在所有的function之外,使用var关键字去声明
3.数组-Array
	1.什么是数组
		数组是一个用于保存批量数据的数据结构
		数组是按照线性结构来保存数据的
	2.创建数组
		1.创建一个空数组
			var 数组名= []
		2.创建一个数组并初始化元素
			var 数组名=[元素1,元素2,...]
		3.创建一个空数组
			var 数组名=new Array();
		4.创建数组并初始化元素
			var 数组名=new Array(元素1,元素2,...)
		注意
			var 数组名=new Array(5)
		只有一个数字的时候表示初始化的长度
			var 数组名=new Array(length)
		length 表示初始化的长度,length为number类型
	3.数组的使用
		获取或设置数组的元素值,一律用下标
		小标从0开始,到数组元素个数-1结束
		1.更改第一个数组元素
			数组名[0]='新名字'
	4.获取数组的长度
		属性:length
		1.适用场合
			1.能够找到数组中,最新要插入元素的位置
				(即向数组尾部添加元素时使用)
			var names = ['刘德华','张学友','郭富城']
			names[names.length]='黎明';
		2.清空数组
			将数组的length属性设置为0,即表示清空数组中所有的元素
		3.配合循环,遍历数组中的第一个元素
			var arr = ['金正恩','金正日','金日成'];
			//从头遍历到尾
			for(var i=0;i<arr.length;i++){

			}
			//从尾遍历到头
			for(var i=arr.length;i>=0;i--){

			}
		练习:
			1.创建一个包含若干整数的数组,找出该数组中的最大值,并打印
			2.创建一个只包含三个数字的数组,将三个数字按照从小到大的方式存放在数组中,并打印
			3.声明一个数组包含若干数字,从弹框录入,判定是否存在,存在,显示下标,不存在,提示不在
	5.常用API
		1.toString()
			作用:将数组装换成字符串,将装换后的字符串进行返回
		2.join(seperator)
			作用:返回一个由seperator连接的数组元素字符串
		3.reverse()
			作用:反转
			注意:该函数会改变现有数组的结构
		4.sort
			1.作用:
				对现有数组的内容进行排序
				默认是按照元素的unicode码升序排序
			2.注意:
				该函数会改变数组的结构
			3.允许通过自定义的排序规则(排序函数)来指定
	              数字的排序方式
	              语法:arr.sort(排序函数);
	                1.升序的排序函数
	                    function sortAsc(a,b){
	                        return a-b;
	                    }
	                  1)将sortAsc指定为排序函数
	                  2)数组会自动的将相邻的两个数字传递到参数的位置处
	                  3)如果返回值是大于0的数字的话,则交换两个数字的位置,否则不变
	                指定排序函数:
	                    arr.sort(排序函数名);
	                    arr.sort(sortAsc);

	                2.降序的排序函数
	                    function sortDsc(a,b){
	                        return b-a;
	                    }
	                3.使用匿名函数指定排序规则
	                    匿名函数:是一个没有名称的函数,只为某一功能二存在
	                    arr.sort(function(a,b){return a-b;});
