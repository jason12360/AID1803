一.基本数据类型

1.number类型

数字类型,可以表示32位的整数或64位的浮点数

整数:允许表示十进制,八进制,十六进制

  • 十进制:var num=321;
  • 八进制:var num=010;
  • 十六进制: var num=0x1A;
  • 例子

小数:

  • 小数计算法:var num=123.456;
  • 指数计算法:var num=1.5e3; e3等于10的3次方

注意:字符串在使用时必须用''或""引起来

  • 1.查看字符的unicode码

    "\u4e00"~"\u9fa5"
  • 4.转义字符
    • \n:换行

二.数据类型转换

不同的数据类型之间的数据在进行运算时如何转换的

1.隐式转换

1.数字+字符串

将数字先转换成字符串,再进行拼接

var num=15;

var str='18';

var r = num+str;//1518

var r = '15'+18+15//151815

var r = 15+18+'15'//3315

var r = '15'+(18+15)//1533

2.数字+布尔

将布尔类型装换为数字

3.布尔+字符串

将布尔装换为字符串,再拼接

var b = true;

var s = '你好'

var r = b+s

r:'true你好'

注意:

任意类型与字符串做加法运算时,都是将其他类型装换为字符串再拼接

字符串与数字进行运算时

  • 加法:拼接
  • 其余操作(-,*,/,%):先将字符串转换为数字,再进行运算

2.显示转换(转换函数,强制转化)

1.toString()

将任意类型的数据转换为字符串,并返回转换后的结果

语法

var r = 变量.toString();

2.parseInt

作用:将任意类型的数据转化为整数,如果实在无法转换的话,则返回NaN(Not a Number)

语法

var r = partseInt(数据);

ex:

1.var r = parseInt('13.5');//13

2.var r = parseInt('13你好');//13

3.var r = parseInt('你好13');//NaN

示例

3.parseFloat

作用等同于parseInt(),允许转换为小数

var r = parseFloat('35.5abc');//35.5

var r = parseFloat('abc35.5');//NaN

4.Number()

将任意类型的数据转换为数字,只要包含非法字符的话,结果就为NaN

var r = Number('35.5')//35.5

var r = Number('35你好')//NaN

运算符

1.算数运算符

+,-,*,/,%,++,--

++:自增运算符,在自身数据的基础上只做+1运算

--:自减运算符,在自身数据的基础上只做-1运算

ex: var a = 15;

a++;//++做后缀,先使用变量的值,再自增

++a;//++做前缀,对变量先自增再使用

示例

2.关系运算符(比较运算符)

>,<,>=,<=,==,!=,===,!==

10 > 5 :true

'10' > 5: true

如果运算符的两端有一个操作数是数字的话,那么另外一个会自动通过Number()进行转换,然后在进行比较

'10a'< 5:false

'10a'> 5:false

NaN和任意类型除了做'不等于'运算返回值均为false

'张三丰'>'张无忌': false

比较每位字符的unicode码,以不相等的为准,最终比较"三"和"无"的Unicode码(charCodeAt())

'10'>'5':false

'50'>'5':true

===和!==

'5'== 5:true

===:比较数据的数值和数据类型都相等的情况下结果才为真

'5'===5:false

!==:比较数据的数值和数据从类型又要有一个不等,结果就是真

'5'!==5:true

3.逻辑运算符

!:非,对现有条件取反

&&:逻辑与

||:逻辑或

4.位运算符

1.按位与:&

3&5:1

3--011(二进制)

5--101(二进制)

判断一个数字的奇偶性

--任意数字与1做按位与操作,结果为1是奇数,结果为0是偶数

2.按位或:|

任意小数与0做按位或操作,快速取整(抛弃小数位,保留整数位)

3.按位异或:^

两个数字的二进制位进行比较,不同则为1,相同则为0

3^5:110

^的作用:在不借助第三方变量的前提下,交换两个数字

a=a^b,b=b^a,a=a^b

单目运算符/一元运算符:有一个操作数的运算符

++,--,!,typeof

=,-,*,/,%,...,&&,||,&,|

三目运算符/三元运算符:有三个操作输的运算符

?
