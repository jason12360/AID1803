整数(int)
    整数是指不带小数部分的数（自然数，0，负数）
    如：5,9，,5
    整形数的字面值表达方式：
        1. 十进制的表示方式
        2. 二进制的表示方式（0b开头，后跟0,1）
        3. 八进制的表示方法（0o开头,后跟0-7）
        4. 十六进制的表示方法（0x开头，后跟0-9，a-f）

浮点数（float）
    浮点数是带有小数部分的数字（小数部分可以为0）
    表示方法：
        小数表示方法：
            3.14,3.1,0.14
        科学计数表示方法：
            格式：
                小数e/E(正负号)指数（10的多次方）
            如：6.18e-1
                2.9e8

负数 complex
    分为两部分
        实部（real）
        虚部（image）
    负数表示方法
        1+2j

布尔 bool
用来表示真和假两种状态的类型
    True     代表真（条件满足或成立）
    False    代表假（条件不满足）
    注意：
        True的值为1
        False的值为0

空值 None
    None是一个表示不存在的特殊对象
作用：
    用来站位
    变量接触绑定

赋值语句：
    语法 
        变量名=表达式
作用：
    1.没有此变量时，创建一个变量，同时将变量绑定到表达式返回的对象上

    一个变量只能绑定在一个对象上
    两个变量可以绑定到同一个对象上

is/is not 运算符
作用：

语法：
    x is y
    x is not y

    a = 1000

练习：
    将数字3.14用变量pi绑定
    将pi变量转换为整数用变量i绑定
    将pi变量与i变量相减，结果用变量f绑定
    1.判断f是否等于0.14
    2.删除所有变量
    （用交互模式来测试上述问题）
        >>> pi = 3.14
        >>> i = int(pi)
        >>> f = pi -i
        >>> f == 0.14
        False
        >>> f
        0.14000000000000012
        >>> pi
        3.14
        >>> i
        3
        >>> del pi,i,f
        >>> pi
    原因：3.14有可能是二进制中无限不循环小数，二进制转换为十进制时会存在精度的问题
