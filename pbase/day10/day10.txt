函数 function
作用：
定义用户级别的函数
什么是函数：
	函数是可以重复执行的代码块，可以重复使用
语法：
	def 函数名（参数列表）：
		语句块（代码块）...
语法说明：
	函数名是语句块的名称
	函数名的命名规则与变量名相同
	函数的名字是一个变量，是绑定代码块的名称
	函数有自己的名字空间，要让函数处理外部数据需要用参数给函数传入一些数据，如果不需要传入参数，参数列表可以为空
	语句块部分不能为空，如果为空需要填充pass语句
函数的调用：
	函数名(实际参数)
调用说明：
	1.函数调用是一个表达式
	2.如果没有return语句，函数执行完毕返回None值对象
	3.如果函数需要返回其他的值对象需要用return语句

return语句
语法：
	return [表达式]
	[]代表可以省略
作用：
	结束当前函数的执行，返回到调用该函数的地方，同时返回一个值对象的引用关系
说明：
	1.return语句后跟的表达式可以省略，省略后相当于return None
	2.如果函数内没有return语句，则函数执行完最后一句返回None（相当于在最后加了一条return None 语句）
	3.函数调用可以返回一个值或一组值
练习：
定义两个函数：
	sum3(a,b,c)用于返回三个数的和
	pow3(x) 用于返回x的立方
	用以上两个函数计算：
		1.计算1的立方+2的立方+3的立方
		2.计算1+2+3的和的立方

函数的参数传递：
传递方式：
	1.位置传参
	2.*序列传参
	3.**关键字传参
位置传参：
	实际参数的对应关系与形式参数的对应关系是以位置来一次对应的
示例：
	def sum3(a,b,c)
		pass
	#        ^ ^ ^	
	x = sum3(1,2,3)
位置传参说明：
	实参和形参通过位置进行对应和传递
	实参和形参的个数必须完全相同

序列传参：
	序列的元素个数必须与列表的个数相同
示例
	见 args.py

关键字传参：
	关键字传参是指传参时，按照形参的名称给形参赋值，实参和形参按名称进行匹配

示例：
	见arg2.py

字典关键字传参
	实参和形参通过字典进行传递和匹配，字典的值传递给建对应的形参
说明：
	字典传参的键名和形参名必须一致
	键名必须为字符串
	键名要在形参中存在

	以下说的是函数形参
函数的缺省参数
语法：
	def 函数名（形参1=默认参数1，形参2 = 默认参数2，...):
		语句
例：
	def info(name, age=1, addr='不详')：
		print('姓名:',name,'年龄:',age,'住址:',addr)
	#调用时
	info('jason','13,'chaoyang')
	info('tarena',15)
	info('jack')
缺省参数说明：
	1.缺省参数必须从右到左依次存在，如果一个参数有缺省值，则其右侧的所有参数都必须有缺省参数（缺省值）
	2.确性参数可以有0个或多个，甚至全部是有缺省参数
例：
def fn(a,b=10,c):pass 这是错的
def fn(a=1,b,c=3):pass 这是错的

练习：
	写一个函数sum4(a,b,c,d)来计算四个参数的和可以用如下方法调用：
	print(sum4(1,2))
	print(sum4(1.1,1.2,3.3))
	print(sum4(100,200,300,400))

函数的不定长参数：
	不定长参数有两种：
		星号元祖传参
		双星号字典传参

星号元祖传参：
	语法：
	def 函数名（*元祖形参名）：
		语句块

min/max
示例见star_args.py

命名关键字形参：
语法：
	def 函数名（*，命名关键字形参名）：
		语句块
	或
	def 函数名（*args，命名关键字形参名）：
		语句块
示例：	
	见named_args.py

双星号字典形参：
语法：
	def 函数名(**字典形参名)：
		语句
例：
	def myfun(**kwargs):
		print('参数个数：',len(kwarg))
		for k,v in kwarg.items():
			print(k,'->>',v)
	调用
	func(name = 'tarena',age = 15)
	func(a = 1,b ='bbb',c=[1,2,3],d = True)

练习：
	写一个函数minmax有不定长的参数，返回这些参数的最大值和最小值，形成元祖，最小值在前，最大值在后
	调用此函数，得到最大值和最小值并打印出来

函数的参数列表顺序：
	位置形参，缺省形参，星号元组形参，双星号字典形参，命名关键字形参可以混合使用
函数参数自左到右的顺序为：
	位置形参，星号元组形参，命名关键字参数，双星号字典形
例：
	def fn(a,b,*args,c,**kwargs):
		pass
	fn(100,200,300,400,c = 'C',d='D',e = 'E')
